# Yici Liu GitHub Pages Resume

This folder is a static personal resume website. It can be deployed directly with GitHub Pages.

## Files

- `index.html`: website content
- `style.css`: visual style
- `assets/profile.png`: portrait image cropped from the uploaded resume screenshot

## Deployment

1. Create a GitHub repository named `your-username.github.io`.
2. Upload all files in this folder to the repository root.
3. Go to repository `Settings` → `Pages`.
4. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Wait one or two minutes.
6. Visit `https://your-username.github.io`.

## Notes

- This template intentionally does not include your phone number. GitHub Pages sites are public, so publishing a phone number is not recommended.
- Replace the email, research status, publications, and project details with your latest information before deployment.
